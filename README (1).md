# 📱 UniPay — Campus Payment App Prototype

> **Design Thinking & Innovation · Course Project**
> A fully interactive mobile app prototype built with Streamlit

---

## 📲 What's Inside

12 fully interactive screens embedded in a realistic Android phone frame:

| Screen | Description |
|--------|------------|
| 🏠 Home | Wallet balance, QR scan, quick actions, spend widget, announcements |
| 🔐 Login | College email + LinkedIn OAuth student verification |
| 📊 Dashboard | AI spending insights, charts, categories, budget tracker, credit score |
| 🏪 Vendors | Searchable grid with filters, ratings, distances, deals |
| 🔥 Hot Deals | Flash offers with timers and discount badges |
| 🎪 Events | Featured fest + upcoming events carousel with registration |
| 💳 Credit | Wallet balance, micro-loans, repayment schedule, leaderboard |
| 👥 P2P Pay | Pay/request with contacts + working numpad |
| 📈 Vendor Dash | Sales graph, heatmap, top items, settlement details |
| 👤 Profile | Avatar, verified badge, settings, toggles |
| 📷 QR Scanner | Animated scanner with torch/gallery options |
| 📋 Transactions | Full history with category filters |

---

## 🚀 Deploy on Streamlit Cloud (Free, ~2 min)

1. Push this folder to a **GitHub repository**
2. Go to **[share.streamlit.io](https://share.streamlit.io)**
3. Click **New App** → select your repo → main file: `app.py`
4. Click **Deploy** — done!

---

## 💻 Run Locally

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run
streamlit run app.py
```

Open `http://localhost:8501` in your browser.
